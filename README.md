# World Globe Study

スマホファーストの世界地図学習アプリです。`index.html` だけで動くので、ファイルを開くだけで試せます。GitHub Pages、Netlify、Cloudflare Pages などの静的ホスティングに置くと、そのままURLで共有できます。

## できること

- 地球儀をドラッグして回転
- ピンチまたはホイールで拡大縮小
- 国名・首都検索
- 国旗、国名、首都、地域、緯度経度を表示
- 初期画面から「今日の5か国」「やさしいクイズ」「日本の近くの国」へ移動
- 今日の5か国
- やさしい・ふつう・むずかしいの国クイズ
- 国旗から国名、国名から国旗、首都から国名、国名から首都のクイズ
- 間違えた国だけをもう一度出す復習クイズ
- 「アメリカ」「イギリス」「UAE」などの別名検索
- 2国の首都間距離を表示
- 選択中の国をURLハッシュに保存して共有
- 人口、面積、人口密度、言語、通貨、東京からの距離を表示
- 南アフリカなど、首都の役割が分かれる国には首都メモを表示

## ファイル構造

- `index.html`
  - HTML: 画面の部品
  - CSS: スマホ向けレイアウトと見た目
  - JavaScript: 地球儀の描画、国データ、検索、クイズ、距離計算

単体HTMLにしている理由は、共有・移動・バックアップが簡単だからです。外部ライブラリを使っていないため、通信環境に左右されにくい作りです。

## 公開について

このアプリはGitHub PagesだけでURL共有できます。Netlifyは必須ではありません。

Netlifyを使うと、ドラッグ&ドロップ公開、フォーム、リダイレクト、独自ドメイン管理などが便利になります。一方で、このアプリのように単体HTMLを公開するだけならGitHub Pagesで十分です。

## データ確認メモ

- 収録範囲は、国連加盟193か国に、台湾・パレスチナ・バチカン市国を加えた学習用の196項目です。
- 国旗は2文字国コードからUnicodeの国旗絵文字を生成しています。
- 人口、面積、言語、通貨、サブ地域はREST Countriesのデータをもとに、2026年4月時点の固定スナップショットとして収録しています。
- 陸地の形はNatural Earthの軽量な陸地GeoJSONを単一HTML内に埋め込んでいます。
- 赤道ギニアは2026年1月2日の首都変更に合わせて、首都をシウダー・デ・ラ・パスに更新しています。
- 南アフリカ、スリランカ、ボリビアなど、首都の役割が分かれる国は `CAPITAL_NOTES` に補足を入れています。
- 一部の人口や面積は統計更新により変わるため、学習用の目安として扱ってください。

## 重要な計算の考え方

### 1. 緯度経度を画面に変換する

地球儀は「正射投影」という考え方で描いています。正射投影は、地球を遠くから見たときのように、手前半分だけを丸く表示する方法です。

コードでは `project(lat, lon)` が担当しています。

- 緯度 `lat`: 北南の位置
- 経度 `lon`: 東西の位置
- `state.centerLon`: いま画面中央に見えている経度
- `state.centerLat`: いま画面中央に見えている緯度
- `z > 0`: 手前側なので描く
- `z <= 0`: 裏側なので描かない

この `z` 判定があるので、地球儀らしく「裏側の国が見えない」表現になります。

### 2. 大陸を3Dっぽく貼る

`LAND_RINGS` にNatural Earth由来の陸地の形を入れています。最初に2Dの世界地図テクスチャを作り、その後で丸い地球の各ピクセルから緯度経度を逆算して色を取っています。

この処理は `renderGlobeTexture()` が担当しています。

### 3. 2国間距離

距離は `distanceKm(a, b)` で計算しています。使っているのはハーサイン式です。

地球を球として見たとき、2つの緯度経度の間の大円距離を求める式です。実際の地球は完全な球ではないため厳密値ではありませんが、学習用の目安として扱いやすい計算です。

## 国を追加する方法

`COUNTRIES` に1行追加します。

```js
{ cc: "XX", name: "国名", en: "English Name", capital: "首都名", lat: 0.00, lon: 0.00, region: "地域" }
```

- `cc`: 2文字の国コード。国旗表示に使います。
- `name`: 日本語名。
- `en`: 英語名。検索にも使います。
- `capital`: 首都名。
- `lat`: 首都の緯度。
- `lon`: 首都の経度。
- `region`: クイズや表示に使う地域名。

首都が一つに言い切りにくい国は、`CAPITAL_NOTES` に表示名と補足を追加します。検索で拾わせたい別名は `SEARCH_ALIASES` に入れます。
